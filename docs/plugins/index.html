Web Search Plugin Example

// Web Search Plugin for Mita Studio
import { MitaPlugin } from '@mita-studio/sdk';
import axios from 'axios';

export default class WebSearchPlugin extends MitaPlugin {
  name = 'web-search-plugin';
  version = '1.0.0';
  
  initialize() {
    this.registerTool({
      name: 'Web Search',
      icon: 'search',
      action: async (context) => {
        const query = context.input;
        const results = await this.performSearch(query);
        return this.formatResults(results);
      }
    });
    
    // Add a custom UI panel for search history
    this.registerPanel({
      id: 'search-history',
      title: 'Search History',
      render: () => this.renderSearchHistory()
    });
  }
  
  async performSearch(query) {
    // Implementation of search functionality
    const response = await axios.get('https://api.search.com', {
      params: { q: query }
    });
    return response.data.results;
  }
  
  formatResults(results) {
    // Format the results for display
    return results.map(r => `[${r.title}](${r.url})\n${r.snippet}`).join('\n\n');
  }
  
  renderSearchHistory() {
    // Custom UI rendering code
    return `

      ${this.getSearchHistory().map(h => 
        `
${h.query}
`
      ).join('')}
    
`;
  }
  
  getSearchHistory() {
    // Get search history from storage
    return this.storage.get('searchHistory') || [];
  }
}

Plugin Development Workflow

1

Set Up Your Development Environment

Install the Mita Studio SDK and development tools to get started.

npm install -g @mita-studio/cli
2

Create a New Plugin Project

Use the CLI to scaffold a new plugin project with all the necessary files.

mita-studio create-plugin my-awesome-plugin
3

Develop Your Plugin

Write your plugin code using the Mita Studio SDK and APIs.

4

Test Your Plugin

Test your plugin in the Mita Studio development environment.

mita-studio dev
5

Package and Publish

Package your plugin and publish it to the Mita Studio Plugin Marketplace.

mita-studio publish
